## How to run

### Requirement

```
// Build Requirement
Flask : 2.1.2
Torch : 1.11.0

// Run Requirement
Docker : 20.10.16
```

### Build & Run

```
// Running Server
$ sudo git clone {this repo}
$ sudo docker build -t deep_learning_server .
$ sudo docker run -d -p 3000:3000 deep_learning_server
```
